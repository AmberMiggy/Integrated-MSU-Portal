# MSU Integrated Student Portal – Figma Prototype 🎓

A high-fidelity, clickable Figma prototype of a unified student portal for Mindanao State University (MSU), integrating the existing AKAN school website and Google Classroom into a single, cohesive platform.

---

## 📌 Project Overview

MSU students currently juggle two separate platforms daily — the AKAN school website for academic records and Google Classroom for class management. This project proposes a unified portal that brings the core features of both systems under one interface, reducing friction and improving the overall student experience.

> **Note:** This project covers the design phase only. No actual software development is involved.

---

## ✨ Features Covered

### 🏫 AKAN Features
- Enrolled courses & class schedule
- Grades and transcript
- INC grade tracker
- Billing and payments
- Faculty evaluation
- Registration and enrolment management

### 📚 Classroom Features
- Class stream and announcements
- Assignments
- Course materials

### 🔗 Shared / Integration Screens
- Unified student dashboard
- Course detail page (combining AKAN enrolment info and Classroom feed)
- Student profile
- Notifications center

---

## 📦 Deliverables
